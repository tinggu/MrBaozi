# MrBaozi
###一个收集妹纸图片的App，并且用了比较多的Rx库写
###开发目的
为了学习DataBinding, RxJava, RxBinding, Retrofit 2.0等知识,参考了[pretty-girl](https://github.com/Assassinss/pretty-girl)。
